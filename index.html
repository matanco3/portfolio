⚙ הגדרות חיבור

🔗 Google Sheet ID

העתק את ה-ID מה-URL של ה-Sheet:
https://docs.google.com/spreadsheets/d/[זה ה-ID]/edit

📋 הוראות הגדרה

1. פתח את קובץ portfolio_google_sheets.xlsx שהורדת
2. עלה אותו ל-Google Drive (גרור לתוך Drive)
3. לחץ ימני → "פתח עם Google Sheets"
4. עבור ל: קובץ → שתף → שתף עם כולם (צופה)
5. העתק את ה-ID מה-URL
6. הדבק אותו כאן ולחץ שמור
